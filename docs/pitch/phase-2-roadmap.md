# Phase 2 Roadmap

## Q1 — Production hardening
- Real bank SSO integration (OAuth 2.0 / SAML token exchange)
- Redis session cache for chat context
- Push notification nudges via bank notification service
- Vector DB (Pinecone) for full product RAG

## Q2 — Advanced wealth features
- Tax-loss harvesting recommendations
- Joint / family accounts with shared goals
- External account aggregation via Open Banking APIs
- Live market price feeds for holdings

## Q3 — Compliance & scale
- Full audit trail for AI recommendations (regulatory reporting)
- Multi-language support for avatar
- Video avatar (speaking) integration
- RM co-pilot dashboard for human handoff cases

## Q4 — Intelligence upgrades
- ML-based behavioral risk scoring (replace rule engine)
- Predictive life-event triggers (marriage, home purchase)
- Automated portfolio rebalancing execution (not just recommendations)
- Custom thematic portfolios (ESG scoring API)
